# ToonHQ Colors Plus
Userscript that recolors [ToonHQ](https://toonhq.org/groups/) gag/suit bubbles to better show toon status at a glance.

Bubbles will turn green for the following:
- Lv. 7 gags
- Highest cog suit (Hollywood, Big Wig, etc)
- v2.0 suit

If there are no matches, the bubble will turn red.

![Example group](https://raw.githubusercontent.com/powwu/toonhq-colors-plus/refs/heads/main/example.png)

## Installation
Use a userscript add-on of your choice. I recommend [Violentmonkey](https://violentmonkey.github.io/).

Navigate to [toonhq-colors-plus.user.js](https://github.com/powwu/toonhq-colors-plus/raw/refs/heads/main/toonhq-colors-plus.user.js) and click install.

If this fails, click the add-on in your toolbar. Copy the contents of [toonhq-colors-plus.user.js](https://github.com/powwu/toonhq-colors-plus/raw/refs/heads/main/toonhq-colors-plus.user.js) into a new script, or paste the URL as an install.

If it does not work at this point, ensure that your add-on is enabled for the website and has the necessary permissions.
